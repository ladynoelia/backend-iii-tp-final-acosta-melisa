# Programación Backend III

## Entrega final de trabajo práctico

### Enlaces

1. GitHub:
   https://github.com/ladynoelia/backend-iii-tp-final-acosta-melisa.git
2. Docker Hub:
   ladynoelia/serverbackend3:1.0.0

### Uso

Descargar imagen desde Docker Hub:
docker pull ladynoelia/serverbackend3:1.0.0

Una vez iniciado, abrir en el navegador:
http://localhost:8080

Documentación Swagger disponible en:
http://localhost:8080/documentation

Para ejecutar los test:
npm test
